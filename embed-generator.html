Canvas LMS Instructions

  1. In Canvas, go to Files and upload your .gp file
  2. Right-click the file → Copy Link Address
  3. Paste the URL above and click Generate Embed
  4. In your Page/Assignment, click the </> HTML editor button
  5. Paste the embed code and save

Hosting the player: For this to work, the embed.html file must be hosted on a web server accessible to your students. You can host it on your own server, GitHub Pages, or any static hosting service. Update the iframe src URL to point to your hosted embed.html location.

CORS Note: If the file fails to load due to browser security (CORS), the embed player will show an "Upload File Instead" button allowing students to manually select the GP file. For best results, host the GP files on the same server as the embed player.
